# Netflix-Data-Analysis
This project involves analyzing a Netflix dataset using Python (pandas and SQL), cleaning the data, and visualizing insights with Power BI. The objective is to explore content distribution, types, durations, and trends over time.

📊 Key Tasks Performed

✅ Data Cleaning

1. Removed or handled null values

2. Standardized column formats (e.g., date_added, duration)

3. Resolved data type mismatches (e.g., year to integer)

✅ Data Querying

1. Used sqlite3 and pandas.read_sql() to run SQL queries on the dataset

2. Extracted insights like:

3. Number of shows per year

4. Movies vs TV shows distribution

5. Country-wise production frequency

6. Duration patterns by content type

✅ Dashboard (Power BI)

1. Map of content distribution by country 🌍

2. Line/bar chart showing release trends over the years 📈

3. Comparison of durations across content types ⏱

4. Filters by genre, country, and type

🧰 Tools & Technologies Used

1. Python (pandas, sqlite3)

2. Power BI (interactive dashboards)

3. SQL (in-memory + persistent queries)

4. Excel (for initial and final data handling)
